CBT Kabut Digital

Bikin CBT Sendiri, Tanpa Ribet Urus Server

Sistem ujian online lengkap — token peserta, anti-kecurangan, koreksi esai, sampai analisis butir soal — jalan di atas Google Sheets milik Anda sendiri. Isi wizard 5 langkah, unduh file setup, selesai.

Cara Kerja
5 langkah, sekitar 5–10 menit
1
Jenis Institusi
Sekolah, kampus, kursus, perusahaan, pemerintah, atau umum — menentukan label kolom otomatis.
2
Identitas
Nama institusi, logo, dan pesan sambutan yang tampil di halaman login peserta.
3
Konfigurasi Ujian
Token, durasi, judul, batas pelanggaran, acak soal, dan tema warna.
4
Bobot Nilai
KKM dan proporsi nilai Pilihan Ganda vs Esai.
5
Preview & Download
Periksa ringkasan, unduh file setup, tempel ke Apps Script — CBT siap dipakai.
Tutorial Bergambar: Setelah File Diunduh
Geser untuk melihat cara memasang file hasil wizard ke Google Sheet Anda
1Buat Google Sheet Baru
2Buka Ekstensi → Apps Script
3Tambahkan Library CBT-KD (Script ID di komentar file)
4Tempel Isi File yang Diunduh
5Deploy sebagai Web App
6Jalankan Menu "Setup Sheet Awal"

Kenapa CBT Kabut Digital
Dibangun untuk institusi yang tidak punya tim IT sendiri

Tanpa Server Sendiri

Jalan di atas Google Sheets & Apps Script yang sudah dimiliki institusi — tidak perlu sewa hosting atau tim IT.

Aman & Terproteksi

Deteksi pelanggaran, kunci sheet otomatis, token reset khusus untuk peserta bermasalah.

Analisis Otomatis

Statistik nilai, tingkat kesukaran soal, deteksi anomali — langsung tersaji per ujian.

Sebelum mulai, kenalan dulu ya

Wizard ini kami sediakan untuk institusi yang benar-benar membutuhkan CBT — isi data berikut untuk melanjutkan.
Isi Nama dan Email terlebih dahulu untuk mengakses wizard.
Nama dan Email wajib diisi agar tim kami dapat menyesuaikan panduan dan menyampaikan informasi pembaruan yang relevan. Lihat Kebijakan Privasi.
